# Product Spec (MVP)

## Vision

Help Catholics pray the rosary while running or walking by aligning prayer decades to distance progress and pace.

## Primary User Story

As a runner/walker, I want the app to guide me through rosary decades while I move, so I can stay focused on prayer and exercise without manual counting.

## Core Flows

1. Choose mode (1 decade / 5 decades / custom).
2. Set target distance.
3. Start session and grant location permission.
4. Receive decade transition prompts while moving.
5. Finish session and review summary/history.

## MVP Functional Requirements

- Start/stop run session.
- Show live distance, pace, and elapsed time.
- Map rosary progress to distance with adaptive updates.
- Trigger prompts at decade boundaries.
- Save completed sessions locally.

## Non-Goals (MVP)

- Account sync/cloud backup.
- Social features.
- Full prayer audio library and multiple voices.
- Advanced route maps/charts.
